# 三种主流服务端的选择

# Bukkit

Bukkit 及其衍生端（如 Spigot、Paper、Purpur 等），可以安装插件（Plugin），市面上的服务器 99% 都是这个端，因为这个服务端最便于管理，玩家只需要原版客户端就能进入服务器（不像模组服，玩家也要装模组），而且插件可以实现的功能非常多，参考 Hypixel。这个端的泛用性也是最广的，如果你想开服面向大众，选这个就可以。但如果你只是想和好友联机，还是更推荐下面的 Fabric 端。（一些常见插件：登录插件、粘液科技、菜单插件、箱子商店、全球市场等。这几个衍生端里，最推荐使用 Purpur。）

# Fabric

Fabric 端，和好友联机首选。无论是原版生存，还是和朋友玩网上下载的地图，都推荐这个端。可以安装 Fabric 模组。玩生电必选这个，因为 Fabric 没有修改原版特性，并且生电的辅助模组（如 Tweakeroo，MiniHUD，Carpet 等）多多少少都需要服务端也安装这些模组才能正常使用。

# Forge

Forge 端，可以安装 Forge 模组，如果想玩整合包或者模组生存，选这个就对了。（注意区分 Forge 模组和 Fabric 模组，别下错版本了）
