# 介绍

本文章介绍的并非网易我的世界的下载安装，而是 Minecraft Java Edition，也就是 Java 版 Minecraft。

本教程基于 Windows 操作系统进行教学，如果使用的是其他操作系统，可以直接联系我。

本文章介绍的方法无需正版账号，可以免费游玩离线版（盗版），但部分服务器无法进入。

如果想购买正版：https://www.bilibili.com/video/BV1tG411D7Dx/ （支持正版！）

你需要准备：一台电脑，一个鼠标。

# 安装 Java

第一步，安装 Java。前往本站的资源页面下载。

文件夹中有三个版本，如果想玩最新版，下载 jdk17 就可以了。

Java8：1.16.x 及以前的版本
Java16：1.17.x 版本
Java17：1.18.x 及以后的版本

下载完成后双击运行，一路 Next 即可。

# 安装启动器

想要运行 mc，需要一个启动器。首推 PCL2 (Plain Craft Launcher 2)。前往本站的资源页面下载。

PCL2 启动器目前仅支持 Windows 操作系统！

选一个地方，专门用来存放所有 MC 相关的东西。把启动器放进去，右键新建快捷方式，再把快捷方式放到桌面，这样就方便多了。

# 配置游戏

双击打开 PCL2，你的界面应该是这样子的。（版本不同可能造成部分差异，但影响不大）

下面我们进行一些初始化操作。

点击顶部横栏中的设置选项。

接下来，像我这样设置。（如果找不到 Java，点击一下自动搜索就可以了）

注意，版本隔离会使每个版本的文件（如存档、材质包、模组等文件夹）都转移到各自的版本文件夹里。例如，1.13.2 的存档文件夹在 .minecraft\versions\1.13.2\saves ，而 1.16.5 的存档文件夹在 .minecraft\versions\1.16.5\saves ，而不开版本隔离则都在 .minecraft\saves 中，这会导致多个版本共用一个目录，可能会出问题，尤其是模组玩家，不同版本共用一个模组文件夹，太不方便了。所以极力推荐大家开启版本隔离。

设置完成后，顶栏回到启动。

如果你没有正版账号，选择离线并输入用户名，即可离线登录。

# 下载游戏

点击顶栏的下载，选择最新版本 1.19.2。

这个界面看不懂？没事，直接点击开始安装就行了。

回到主界面，点击启动游戏，开启你的 Minecraft 旅程吧。
