Minecraft下载与安装

发表于 2022-12-06
  • Aerco Tavern
首页 Minecraft

本文章介绍的并非网易我的世界的下载安装，而是 Minecraft Java Edition，也就是 Java 版 Minecraft。

本教程基于 Windows 操作系统进行教学，如果使用的是其他操作系统，可以直接联系我。

你需要准备：一台电脑，一个鼠标。

# 安装 Java

第一步，安装 Java。点我下载

文件夹中有三个版本，如果想玩最新版，下载 jdk17 就可以了。

下载完成后双击运行，一路 Next 即可。

# 安装启动器

想要运行 mc，需要一个启动器。首推 PCL2 (Plain Craft Launcher 2)。点我下载

选一个地方，新建一个文件夹，专门用来存放 mc 相关的东西。把启动器放进去，右键新建快捷方式，再把快捷方式放到桌面，这样就方便多了。

# 配置游戏

双击打开 PCL2，你的界面应该是这样子的。

下面我们进行一些初始化操作。

点击顶部横栏中的设置选项。

接下来，像我这样设置。（如果找不到 Java，点击一下自动搜索就可以了）

设置完成后，顶栏回到启动。

如果你没有正版账号，选择离线并输入用户名，即可离线登录。

当然，我还是更推荐花 89RMB 购买正版账号。购买流程看我的另一篇文章。

# 下载游戏

点击顶栏的下载，选择最新版本 1.19.2。

这个界面看不懂？没事，直接点击开始安装就行了。

回到主界面，点击启动游戏，开启你的 Minecraft 旅程吧。
