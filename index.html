Python基础教程第一章——理解Python中的基本语句

# 第一节 如何让程序理解你在说什么

写代码，听上去很高大上，但其实说白了就是命令程序做事。只不过，你直接说中文，他听不懂，你需要用他能理解的方式来命令他。

想命令程序很简单，按照 “主谓宾” 的方式即可。比如，想表达人吃苹果，拆分一下，可以得到：

主语——人
谓语——吃
宾语——苹果

那么如何翻译成 Python 语言呢？如下：

person.eat(apple)

可以看到，主语和谓语之间用 . 来分隔，而宾语是作为谓语的补充成分，放在谓语后边的括号里的。

服务器开服教程

# 前言

有关服务端的选择，已经在另一篇文章详细讲解过了，那么废话不多说，直接开始教学。

# 开服教程

# 下载

首先，下载你需要的服务端。

Fabric：Fabric 服务器下载

Forge：Forge 官方网站（Forge 下载过程中，如果在广告页面卡住，需要翻墙，如果没有翻墙工具请通过其他方式获取，比如镜像源，也可以直接找我）

Paper：Paper 服务器下载

Purpur：Purpur 服务器下载

Vanilla（原版服务器）：原版服务器下载

Spigot：Spigot 服务器下载

# 准备工作

首先，新建一个文件夹，作为服务器文件夹。然后，将下载的 jar 文件放到服务器文件夹里。

如果使用的是Forge服务端，请直接跳过下面的内容，查看“Forge开服教程”的内容。

接下来，新建一个文本文档，连名带后缀一块改成 start.bat （如果后缀名不显示，请先开启文件后缀显示，方法自行百度）。

随后，右键 start.bat ，点击 编辑 ，输入以下内容：

java -Xms2G -Xmx2G -jar server.jar nogui
pause

什么意思呢？一个一个看。

服务端的选择

# 省流

Bukkit 及其衍生端（如 Spigot、Paper、Purpur 等），可以安装插件，如果你想玩生存，选这个就可以。（还可以安装插件，比如登录插件、粘液科技等等。这几个衍生端里，最推荐使用 Purpur。）

Fabric 端，可以安装 Fabric 模组，玩生电必选这个，因为 Fabric 没有修改原版特性，并且生电的辅助模组（如 Tweakeroo，MiniHUD，Carpet 等）多多少少都需要服务端也安装这些模组才能正常使用。

Forge 端，可以安装 Forge 模组，如果想玩整合包或者模组生存，选这个就对了。（注意区分 Forge 模组和 Fabric 模组，别下错版本了）

如果你下载了网上的地图想和朋友联机，推荐直接使用 Minecraft 原版服务端，因为 Bukkit 端某些版本的存档保存方式与传统方式不一样，可能地图无法正常加载。（Dim-1 和 Dim1 从 world 文件夹单独分到 world_nether 和 world_the_end 文件夹了）

方块竞速游戏介绍

# 方块竞速 BlockRacing

这是一个 Minecraft 多人竞速小游戏，分为两个队伍，先收集完指定方块的队伍获胜。

版本：Java 1.19.3

条件允许可以去 Github 看

# 特色功能

  1. 队伍箱子：每个队伍都有 3 个队伍箱子，箱子里的物品对同队伍成员共享。
  2. 奖励机制：每收集一个方块，对方队伍将会获得一组该方块（存放在队伍箱子里）。
  3. 队伍 TP：同队伍之间可以自由 TP。
  4. Roll：当前所需方块太难获取时，可以花费队伍积分轮换掉，每局仅限一次。
  5. 定位：玩家可以花费队伍积分购买 locate 指令，用于定位群系或结构。
  6. 记录点：每个队伍有 3 个记录点，可以自由保存、传送、删除。
