# 第一节 如何让程序理解你在说什么

写代码，听上去很高大上，但其实说白了就是命令程序做事。只不过，你直接说中文，他听不懂，你需要用他能理解的方式来命令他。

想命令程序很简单，按照 “主谓宾” 的方式即可。比如，想表达人吃苹果，拆分一下，可以得到：

主语——人
谓语——吃
宾语——苹果

那么如何翻译成 Python 语言呢？如下：

person.eat(apple)

可以看到，主语和谓语之间用 . 来分隔，而宾语是作为谓语的补充成分，放在谓语后边的括号里的。

再来看几个例子，找一找规律：

rabbit.jump(highly)
kid.run(fast)
bird.fly()

在写代码时，如果想让程序做一件事，动词后边必须带有括号。比如上面的 eat、jump、run 和 fly，后边就必须加括号。括号里可以补充这个动词的细节，也可以不写。（此处的谓语并非严格意义上的谓语，只是方便你理解而已）

如果一个语句的主语是程序本身，那么主语可以省略不写。比如，如果你想让 Python 打印（输出）一段文字到控制台，你可以直接写：

print("一段文字")

print 是动词，意为打印，后边的括号里填写要打印的内容，这就是一个最基本的语句。

当然，Python 能理解的动词也是有限的，像上面那些 eat ()、jump ()、run ()、fly () 等等，都是我编出来的，Python 是不认识的。你需要学习的，是这些语句的表达方式。

这时候你肯定想问：那主语都有谁呢？

其实，你需要学的内容中，主语 99% 都是程序本身，所以放轻松，不用写主语，直接写动词和括号就可以了。

有点晕乎乎的？没关系，在后边的学习中，你会越来越熟悉的。

# 第二节 两种数据类型

我们重新看一下上边的那行代码：

print("一段文字")

你会发现，括号中还有一对引号。这是为什么？

引号包起来的内容，称为字符串。按照字面意思，很好理解，字符组成的串，就是字符串。说白了，就是很多个文字或者符号组成的一串东西。引号里可以是中文、英文、数字或者字符，随意组合，没有限制（除了反斜杠 \ 和引号不能乱加）。

有多随意呢？看下面这几行，都是字符串：

"Hello World!"
"KFC疯狂星期四V我50"
"N*lP05xsVhMX@$PAHY-a"
"为 什 么 你 们 打 字 要 加 空 格"
""

可以看到，只要在引号之内，写啥都行。程序不会试图去理解引号内部你写了什么，程序只知道，这是一个字符串，存放着一堆字符。（两个引号中间可以什么都不加，这是一个空字符串，虽然里边没有存放任何东西，但他确实是一个字符串）

但是，Python 中可不止这一种存放数据的方式。

如果一个数据，全部都是整数数字，那么就不用加引号了。这是不同于字符串的另一种数据类型，称为整数类型。比如：

123456789
114514
-250

一个整数类型中的数字也是有大小限制的，他的范围是 -2147483648~2147483647 。

这时候你肯定想问：如果一个数字加了引号，他是什么类型的呢？

"114514"

根据上面学的知识：** 只要加了引号就是字符串，所有的字符串都带引号。** 所以，这是一个字符串。

# 第三节 变量

在实际开发中，数据很多很多，而且经常变化，很难利用。不过，Python 允许我们创建一个小盒子，把数据装进去之后，给盒子起个名字，这样以后想要使用某一个数据时，直接找指定的盒子就可以了。这个小盒子，就是变量。

那如何创建一个变量呢？如下：

age = 10

可以看到，左边是一个单词 age ，中间是一个等号，右边是一个整数类型的数字 10 。

左边的单词 age ，其实就是小盒子的名字。这行代码的意思是，把数据 10 装到一个名为 age 的小盒子中。这就是给变量赋值。

赋值的语法很简单：

变量的名字 = 要存入的数据

看下面几个例子巩固一下：

name = "小明"
age = 10
sex = "男"
height = 140

存放好了数据，该怎么使用呢？直接拿变量名就可以替代数据本身，如下：

print(name)

输出结果为：

小明

可以看到，括号里填入 name 这个变量名时，由于这个变量中存放的数据是 "小明" ，所以这个数据直接被取出来用了。

那如果，我就想输出 name 这个英文单词，怎么办？

加个引号，变成字符串，不就行了：

print("name")

输出结果为：

name

有点理解了吧？我们来个实战！

让用户输入两个数字，Python 需要对两个数字进行求和，然后把结果输出。

梳理一下思路：

用户输入第一个数字，储存在变量num1中
用户输入第二个数字，储存在变量num2中
将num1与num2求和，将结果储存在变量answer中
输出变量answer

完了，第一行就不会了。用户怎么输入数字？

这里，需要使用 input() 。直译过来，就是输入的意思。当程序执行到这里时，会打开一个输入框让用户输入东西，输入完成之后回车，程序就会把用户输进去的东西转换成字符串传到程序当中。我们可以用一个变量来储存用户输入的内容，如下：

a = input("请输入内容")

括号里可以填写文字，这些文字会在输入框上方呈现。你可以写一些注解，告诉用户需要输进去什么东西。

那么完整的代码就写出来了：

num1 = input("请输入第一个数字")
num2 = input("请输入第二个数字")
answer = num1 + num2
print(answer)

很好理解吧！

诶，但我可以告诉你，这段代码的表述是有问题的。

如果你尝试去运行，你会得到一个奇怪的结果：如果用户输入的数字是 10 和 20 ，输出的结果为 1020 。

哪里出错了？

其实这是 Python 中很坑的一个地方。 input() 返回的内容，是一个字符串。假如在第一行中，用户输的是 10 ，那么实际储存在 num1 中的东西，是 "10" 。然而，字符串是无法进行数学运算的。在 Python 中，如果两个字符串相加，Python 会将这两个字符串进行拼接，这才有了上面的结果。如果你想进行数学运算，必须使用两个整数类型的数据（还可以用浮点型或布尔型，后边会讲）。

解决方法也很简单，将这个字符串 "10" 转换成整数类型 10 即可。

我们可以使用 int() 来进行转换，如下：

int_num1 = int(num1)

注意，在赋值时，Python 都会先执行等号右边的内容，然后再把结果赋值到等号左边。

因此，Python 会先执行 int(num1) ，将 num1 中的字符串类型的数据转换成整数类型的数据，然后再赋值到左边的 int_num1 。

现在， int_num1 中存放的数据就是 10 了。

刚才的代码修改如下：

num1 = int(input("请输入第一个数字"))
num2 = int(input("请输入第二个数字"))
answer = num1 + num2
print(answer)

可以看到， input() 被包在了 int() 的括号中，用户输入的结果被直接转换成了整数类型。现在，代码的运行就没有问题了。快去试试吧！

# 第四节 拓展

这一节是拓展内容。如果你自认为学的还算明白，那么可以看一看下面的内容加深巩固。

在字符串那里，我提到过一个注意事项，字符串引号内不能加反斜杠 \ 和引号。为什么？

这就涉及到反斜杠的一个重要作用了：转义。

如果你想在字符串中加入换行，该怎么办？肯定不能直接换行啊，Python 不允许这样做。

但是， \n 可以表达换行。在字符串内加入 \n ，就表明这里有一个换行。如下：

"我是第一行\n我是第二行"

如果你想在字符串内部加引号，不能直接加，不然 Python 就不知道字符串的起始位置了。你可以在引号前添加转义字符，这样 Python 就知道，这个引号时字符串里的东西，而不是字符串的边界。如下：

print("他说：\"你干嘛哈哈哎呦\"")

输出结果：

他说："你干嘛哈哈哎呦"

想了解更多，可以看百度百科中对转义符的介绍：转义符

第二个要拓展的内容，就是格式化字符串。

直接看例子：

name = "小明"
age = 10
print(f"我的名字是{name}，我的年龄是{age}岁")

在 print() 中，有一个字母 f 和一个字符串。当字母 f 紧贴在一个字符串前，字符串内可以用花括号来包住变量名，比如 {name} 在输出时就会变成 "小明" 。输出结果如下：

我的名字是小明，我的年龄是10岁

格式化字符串有很多种方式可以实现，这是我认为最清晰明了的一种，也是最好记的一种。
